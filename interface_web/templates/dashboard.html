Argument Quality Evaluation

Evaluates arguments on 9 dimensions: clarity, coherence, relevance, sufficiency, accuracy, depth, originality, fairness, and persuasiveness.

Counter-Argument Generation

Generates counter-arguments using 5 strategies: reductio ad absurdum, counter-example, distinction, reformulation, and concession+pivot.

Fallacy Detection

Detects fallacies using 3-tier hybrid detection: heuristic rules, CamemBERT classifier, and 8-family taxonomy (Ad Hominem, Red Herring, False Cause, etc.).

Adversarial Debate

Agent A defends the strongest arguments while Agent B attacks using identified weaknesses. A judge evaluates each exchange.

Collaborative Multi-Agent Analysis

Four specialized agents collaborate sequentially: Critic (identifies weaknesses), Validator (checks evidence), Devil's Advocate (generates counter-arguments), and Synthesizer (resolves disagreements).

Governance Simulation

Applies 7 voting methods (majority, Borda, Condorcet, approval, etc.) to evaluate competing arguments and reach a collective decision.

JTMS / Logic Reasoning

The JTMS (Justification-based Truth Maintenance System) provides belief management with dependency tracking and consistency checking.
