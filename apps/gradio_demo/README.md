# Gradio Demo — Live Argumentation Analysis

Single-page ML-style demo for soutenance. No build step required.

## Launch

```bash
# Install Gradio (not in main deps)
pip install gradio

# Set API key
export OPENAI_API_KEY=sk-...

# Run
python apps/gradio_demo/app.py
# Open http://localhost:7860
```

## Features

- **Text input**: Paste any argumentative text
- **Sample scenarios**: Load pre-built political, scientific, media, or philosophical texts
- **Workflow selection**: light (5 phases, ~15s), standard (10 phases), spectacular (17 phases, ~45s)
- **Tabbed output**:
  - Arguments extracted with roles
  - Detected fallacies with confidence scores
  - Formal logic results (PL, FOL, Modal, Dung, JTMS, ATMS)
  - Counter-arguments generated by 5 rhetorical strategies
  - Quality scores per argument
  - Narrative synthesis

## Troubleshooting

| Issue | Fix |
|-------|-----|
| `ModuleNotFoundError` | Run from repo root, or set `PYTHONPATH` |
| `OPENAI_API_KEY not set` | Export the key or add to `.env` |
| Timeout | Use `light` workflow for faster results |
| Empty tabs | Some phases may fail — check status message |
| `gradio not found` | `pip install gradio` |

## Architecture

The app calls `run_unified_analysis()` from `argumentation_analysis/orchestration/unified_pipeline.py`, the same pipeline used by the CLI, API, and batch runner. No separate code path.
