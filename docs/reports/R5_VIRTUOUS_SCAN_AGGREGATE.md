## Caveat (load-bearing)

The lexical screen is a **recall-oriented candidate generator**. Its PRECISION
is unknown: a long text scoring `density 0.0` can be structurally fallacious
(non sequitur, circularity) or simply in an unmatched language/register. **Do
not assert a candidate is virtuous without a pipeline run** confirming
`fallacy_count ≈ 0 AND a non-trivial formal/quality axis (spec §5.1).** The
next step is po-2025's budget-gated capstone run (#1160) on the top candidates
to derive the flag via `detect_virtuous_mode`.

## Reproduction (deterministic, 0 LLM/JVM/API)

```bash
conda run -n projet-is --no-capture-output python scripts/scan_virtuous_corpus.py
```

Re-runnable; bounds overridable via `--min-chars/--max-chars/--max-density`.
The script exits non-zero (`2`) when the pool is THIN (≤2 candidates) — the
rarity fail-loud guard, so a caller/CI cannot mistake a thin pool for success.
